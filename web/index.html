QuickEd Web

Welcome to QuickEd Web.
This is a simple web interface to get the alignment score and CIGAR string of two DNA sequences, using the QuickEd library.
Check the QuickEd repository for more information on the library.

This interface uses the QuickEd library through WebAssembly. Everything is done in your browser, no data is sent to any server.

Warning: This is a simple interface for demonstration purposes. It is heavily experimental and heavily undertested, so it may not work as expected in all cases. Shoot the messenger in this case, the message is sound 😉
